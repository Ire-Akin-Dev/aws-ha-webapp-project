\# Highly Available Web Application on AWS



This project demonstrates the design and deployment of a highly available and secure web application architecture on AWS.



\## Architecture Highlights



\- Custom VPC with public and private subnets

\- Multi-AZ deployment

\- Application Load Balancer

\- EC2 instances in private subnets

\- NAT Gateway for outbound internet access

\- Secure security group architecture

\- Health checks and target groups

\- Highly available design



\## Technologies Used



\- Amazon EC2

\- Application Load Balancer

\- VPC, Subnets, Route Tables

\- NAT Gateway \& Internet Gateway

\- Security Groups \& IAM

\- Amazon Linux 2023
